## Usage
### Example

[`.scaffold`](./.scaffold)

```
$ tree .scaffold
.scaffold
├── command
│   ├── app
│   │   └── cmd
│   │       ├── {{name}}.go
│   │       └── {{name}}_test.go
│   └── meta.toml
└── usecase
    ├── app
    │   └── usecase
    │       ├── {{name}}.go
    │       └── {{name}}_test.go
    └── meta.toml

$ scaffold g command destroy
       exist  .
       exist  app/cmd
      create  app/cmd/destroy.go
      create  app/cmd/destroy_test.go

$ ls app/cmd/destroy*
app/cmd/destroy.go
app/cmd/destroy_test.go
```


### Available filters

- `toUpper`
- `toLower`
- `camelizer`
- `pascalize`
- `underscore`
- `dasherize`


## Installation
### macOS
Homebrew formula is available at [izumin5210/homebrew-tools](https://github.com/izumin5210/homebrew-tools)

```
$ brew install izumin5210/tools/scaffold
```

### Other platforms
Prebuilt binary is available at [Releases](https://github.com/izumin5210/scaffold/releases)

## Development

```
# install dependencies
$ make deps

# build
$ make build

# run lint and test
$ make test
```
